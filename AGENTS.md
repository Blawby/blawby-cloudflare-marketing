# AI Agent Instructions

This project uses specific rules for AI agents to follow when generating or modifying code and content.

Please refer to the following rules based on the task you are performing:

- **[Site Rules](agents/blawby-marketing-site-rules.md)**: Core features, React component rules, SEO requirements, and general development guidelines for the Blawby marketing site.
- **[Content Strategy](agents/content-strategy.md)**: Audience personas, content buckets, publishing cadence, internal linking, and SEO strategy for blog posts.
- **[Writing Rules](agents/writing-rules.md)**: Tone and voice, MDX formatting, and structural requirements for LLM-generated blog posts.
- **[AI Chat & Conversation Rules](agents/ai-chat-conversation-rules.md)**: Guidelines for documenting AI chat features, conversation management, client intake automation, and real-time messaging functionality.
- **[Technical Documentation Rules](agents/technical-docs-rules.md)**: API documentation standards, webhook setup guides, integration examples, and developer-focused content requirements.

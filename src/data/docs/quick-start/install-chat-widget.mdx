# Install the Chat Widget

Adding the Blawby chat widget to your website helps your firm capture leads 24/7, qualify prospects automatically, and reduce intake friction. With the widget, potential clients can start intake instantly—no phone tag or email delays—so you never miss a lead, even after hours.

Install your chat widget so potential clients can complete intake directly from your website. For more on intake workflows, see [Intake Workflows Guide](/ai-chat/modern-intake-flow).

## Section 1: The easy way (no technical skills needed)

If you are a lawyer or paralegal and do not manage website code, start here.

### Find your chat URL in Blawby

1. Sign in to your Blawby dashboard.
2. Open your widget/chat setup area.
3. Copy your chat URL.

### Use your chat URL immediately

You can launch intake today without touching website code:

- Paste the chat URL in intake emails
- Add it to your email signature
- Share it with potential clients directly

### Ask your web designer to add it to your website

If someone else manages your site, forward them this page and point them to [Section 3: Developer installation](#section-3-developer-installation).

## Section 2: Add it to your website yourself

If your website uses a builder, you can usually add the widget without coding.

### Squarespace

1. Open the page where you want intake chat available.
2. Add an **Embed** block.
3. Paste the widget code from your Blawby setup.
4. Save and publish.

### Wix

1. Open your site editor.
2. Add an **Embed** element.
3. Paste the widget code from Blawby.
4. Save and publish.

### WordPress

1. Open the page editor.
2. Add a **Custom HTML** block.
3. Paste the widget code from Blawby.
4. Update/publish the page.

### Quick test after publishing

1. Open your website in a private browser window.
2. Confirm the chat appears.
3. Send a test message.

## Section 3: Developer installation

Use this section if your web developer wants script-level installation.

```html
<script>
  (function (w, d, s, o, f, js, fjs) {
    w["BlawbyWidget"] = o;
    w[o] =
      w[o] ||
      function () {
        (w[o].q = w[o].q || []).push(arguments);
      };
    js = d.createElement(s);
    fjs = d.getElementsByTagName(s)[0];
    js.async = 1;
    js.src = f;
    fjs.parentNode.insertBefore(js, fjs);
  })(
    window,
    document,
    "script",
    "blawby",
    "https://widget.blawby.com/embed.js",
  );

  blawby("init", {
    practiceId: "YOUR_PRACTICE_ID",
    position: "bottom-right",
    theme: "light",
  });
</script>
```

Replace `YOUR_PRACTICE_ID` with your practice ID from Blawby.

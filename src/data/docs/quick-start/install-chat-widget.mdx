# Install the Chat Widget

Add the Blawby AI chat widget to your website in minutes. This guide covers all major platforms and installation methods.

## What You'll Need

- Your Blawby practice ID (from your dashboard)
- Access to your website's code or CMS
- Basic understanding of HTML (no coding experience required)

## Option 1: WordPress (Recommended)

### Plugin Installation
1. Log into your WordPress admin dashboard
2. Go to **Plugins → Add New**
3. Search for "Blawby Chat Widget"
4. Click **Install Now**, then **Activate**

### Configuration
1. Go to **Settings → Blawby Chat**
2. Enter your Practice ID (found in your Blawby dashboard)
3. Choose your widget position and appearance
4. Click **Save Changes**

### Customization Options
- **Position**: Bottom-right, bottom-left, top-right, top-left
- **Theme**: Light, dark, or auto (matches user preference)
- **Colors**: Match your brand colors
- **Welcome Message**: Custom greeting for visitors

## Option 2: HTML/CSS Websites

### Basic Installation
Add this code before your closing `</body>` tag:

```html
<script>
  (function(w, d, s, o, f, js, fjs) {
    w['BlawbyWidget'] = o;
    w[o] = w[o] || function() {
      (w[o].q = w[o].q || []).push(arguments)
    };
    js = d.createElement(s);
    fjs = d.getElementsByTagName(s)[0];
    js.async = 1;
    js.src = f;
    fjs.parentNode.insertBefore(js, fjs);
  })(window, document, 'script', 'blawby', 'https://widget.blawby.com/embed.js');
  
  blawby('init', {
    practiceId: 'YOUR_PRACTICE_ID',
    position: 'bottom-right',
    theme: 'light'
  });
</script>
```

Replace `YOUR_PRACTICE_ID` with your actual practice ID from the Blawby dashboard.

### Advanced Customization
```html
<script>
  blawby('init', {
    practiceId: 'YOUR_PRACTICE_ID',
    position: 'bottom-right',
    theme: 'light',
    customColors: {
      primary: '#1e40af',    // Your brand blue
      secondary: '#3b82f6',  // Lighter blue
      accent: '#f59e0b'      // Accent color
    },
    welcomeMessage: 'Hello! How can I help you with your legal needs today?',
    showBranding: false,
    autoOpen: false,
    mobileOptimized: true
  });
</script>
```

## Option 3: React Applications

### Installation
```bash
npm install @blawby/widget
```

### Basic Usage
```jsx
import { BlawbyWidget } from '@blawby/widget';

function App() {
  return (
    <div>
      {/* Your existing app content */}
      <BlawbyWidget 
        practiceId="YOUR_PRACTICE_ID"
        position="bottom-right"
        theme="light"
      />
    </div>
  );
}
```

### Advanced Configuration
```jsx
import { BlawbyWidget } from '@blawby/widget';

function App() {
  const widgetConfig = {
    practiceId: 'YOUR_PRACTICE_ID',
    position: 'bottom-right',
    theme: 'light',
    customColors: {
      primary: '#1e40af',
      secondary: '#3b82f6'
    },
    welcomeMessage: 'Welcome to our law firm! How can we help you today?',
    onConversationStart: (conversationId) => {
      console.log('New conversation started:', conversationId);
    },
    onMessageReceived: (message) => {
      console.log('New message:', message);
    }
  };

  return (
    <div>
      <BlawbyWidget {...widgetConfig} />
    </div>
  );
}
```

## Option 4: Other Website Builders

### Squarespace
1. Go to **Settings → Advanced → Code Injection**
2. Add the widget code to the **Footer** section
3. Save and publish your site

### Wix
1. Go to **Settings → Tracking & Analytics**
2. Click **New Tool → Custom**
3. Paste the widget code in the **Body** section
4. Save and publish your site

### Shopify
1. Go to **Online Store → Themes**
2. Click **Actions → Edit Code**
3. Open `theme.liquid`
4. Add the widget code before `</body>`
5. Save the file

### Webflow
1. Go to **Project Settings → Custom Code**
2. Add the widget code to the **Footer Code** section
3. Save and publish your site

## Widget Configuration Options

### Position Settings
- `bottom-right` (default) - Most common, least intrusive
- `bottom-left` - Alternative for left-aligned designs
- `top-right` - For header placement
- `top-left` - For top-left corner placement

### Theme Options
- `light` - Clean, professional appearance
- `dark` - Modern, sleek design
- `auto` - Automatically matches user's system preference

### Custom Styling
```javascript
blawby('init', {
  practiceId: 'YOUR_PRACTICE_ID',
  customColors: {
    primary: '#1e40af',      // Main brand color
    secondary: '#3b82f6',    // Secondary color
    accent: '#f59e0b',       // Accent/highlight color
    background: '#ffffff',   // Chat background
    text: '#1f2937'          // Text color
  },
  customFont: 'Inter',       // Custom font family
  borderRadius: '8px',       // Border radius
  shadow: '0 4px 6px rgba(0, 0, 0, 0.1)' // Box shadow
});
```

### Behavior Settings
```javascript
blawby('init', {
  practiceId: 'YOUR_PRACTICE_ID',
  autoOpen: false,           // Don't automatically open
  delayOpen: 5000,          // Open after 5 seconds
  showBranding: true,        // Show "Powered by Blawby"
  mobileOptimized: true,     // Optimize for mobile devices
  soundEnabled: false,       // Disable notification sounds
  persistentChat: true       // Remember conversation history
});
```

## Testing Your Installation

### Basic Test
1. Open your website in a browser
2. Look for the chat widget in your chosen position
3. Click the widget to open it
4. Send a test message

### Advanced Testing
1. **Cross-browser testing**: Chrome, Firefox, Safari, Edge
2. **Mobile testing**: iPhone, Android, tablets
3. **Responsive design**: Different screen sizes
4. **Functionality**: Message sending, file uploads, etc.

### Troubleshooting Common Issues

**Widget Not Appearing**
- Check your Practice ID is correct
- Verify the script loads without errors (check browser console)
- Ensure your website allows third-party scripts

**Widget Positioned Incorrectly**
- Try a different position setting
- Check for CSS conflicts on your website
- Test on different screen sizes

**Messages Not Sending**
- Verify your practice is active and in good standing
- Check your internet connection
- Ensure the widget is properly authenticated

**Styling Issues**
- Disable custom CSS temporarily
- Try default theme settings
- Check for conflicting stylesheets

## Mobile Optimization

The widget automatically optimizes for mobile devices, but you can customize the experience:

```javascript
blawby('init', {
  practiceId: 'YOUR_PRACTICE_ID',
  mobileOptimized: true,
  mobileSettings: {
    position: 'bottom-center',  // Better for mobile
    fullScreenOnOpen: true,     // Open in full screen on mobile
    swipeToClose: true,         // Allow swipe gestures
    virtualKeyboardOptimized: true
  }
});
```

## Analytics and Tracking

### Built-in Analytics
The widget automatically tracks:
- Conversation start rate
- Message volume
- Lead conversion rate
- User engagement metrics

### Custom Event Tracking
```javascript
blawby('init', {
  practiceId: 'YOUR_PRACTICE_ID',
  tracking: {
    googleAnalytics: true,      // Send events to GA
    customEvents: true,         // Enable custom events
    trackPageViews: true       // Track page views
  },
  onConversationStart: (data) => {
    // Send to your analytics
    gtag('event', 'conversation_start', {
      practice_id: data.practiceId,
      source: data.source
    });
  }
});
```
